# Grocery-Store-War-Zepto-vs-Blinkit-Dashboard
This project presents an interactive Power BI dashboard comparing the sales performance of Zepto and Blinkit, two leading quick-commerce platforms in India.
The dashboard enables side-by-side analysis of sales, orders, delivery time, customer ratings, categories, and cities to understand competitive performance and business trends.

The project demonstrates skills in data analysis, visualization, KPI design, and business storytelling using Power BI.
